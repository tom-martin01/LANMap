LANMap
--------------------------------------------------------------------------------
Description: A Local Area Network visualisaiton tool developed in Python.
             Similar to Zenmap Graphical User Interface program.
Learning Aims: Python3, Networking, GUI for Python, Graphing, Data Analysis.
Requirements:
  - Scan LAN for devices.
  - Display key Network Information for each device.
      - IP Address
      - MAC Address
      - Type of Device
      - Vendor
      - Open Ports
  - Display discovered devices in a interactive network graph.
  - Display visual traceroute flow to target.
  - Displayed using Network related Icons.
  - Ability to Import and Export files (JSON format?).
  - Ability to customise visualisaiton details on graph.
  - Housed within a GUI.
  - Designed for Unix.
